**Storage crate:**
- Replace `BlufioError::Storage { source }` constructions with typed StorageErrorKind:
  - SQLITE_BUSY errors: `BlufioError::storage_busy(err)` (maps to FailureMode::Unavailable, retryable)
  - Corruption/integrity errors: `BlufioError::storage_corruption(err)`
  - Schema migration failures: `BlufioError::Storage { kind: StorageErrorKind::SchemaError, ... }`
  - Disk full: `BlufioError::Storage { kind: StorageErrorKind::DiskFull, ... }`
  - Connection failures: `BlufioError::Storage { kind: StorageErrorKind::ConnectionFailed, ... }`
  - Generic/unknown storage errors: default to StorageErrorKind::ConnectionFailed with source
- Remove all `storage_generic()` deprecated fallback calls.

**MCP client crate:**
- Replace `BlufioError::Mcp { message, source }` with typed McpErrorKind:
  - Connection errors: `BlufioError::mcp_connection_failed(err)`
  - Tool execution failures: `BlufioError::mcp_tool_failed(err)`
  - Timeout: `BlufioError::Mcp { kind: McpErrorKind::Timeout, ... }`
  - Auth failures: `BlufioError::Mcp { kind: McpErrorKind::AuthFailed, ... }`
  - Protocol errors: `BlufioError::Mcp { kind: McpErrorKind::ProtocolError, ... }`
- Remove all `mcp_generic()` deprecated fallback calls.

**Skill crate:**
- Replace `BlufioError::Skill { message, source }` with typed SkillErrorKind:
  - Execution failures: `BlufioError::skill_execution_failed(err)`

  - Sandbox timeout: `BlufioError::Skill { kind: SkillErrorKind::SandboxTimeout, ... }`
  - Compilation failures: `BlufioError::Skill { kind: SkillErrorKind::CompilationFailed, ... }`
- Remove all `skill_generic()` deprecated fallback calls.

  - Version mismatch: `BlufioError::Migration { kind: MigrationErrorKind::VersionMismatch, ... }`
