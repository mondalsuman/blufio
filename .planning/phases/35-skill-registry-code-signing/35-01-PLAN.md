**signing.rs:** Create the signing module following DeviceKeypair patterns:

```rust
use ed25519_dalek::{Signature, Signer, SigningKey, Verifier, VerifyingKey};
use rand::rngs::OsRng;
use sha2::{Sha256, Digest};
use blufio_core::BlufioError;
```

1. `pub struct PublisherKeypair` with `signing_key: SigningKey` and `verifying_key: VerifyingKey`.
   - `pub fn generate() -> Self` — `SigningKey::generate(&mut OsRng)`, derive verifying key.
   - `pub fn from_bytes(private_bytes: &[u8; 32]) -> Result<Self, BlufioError>` — reconstruct from stored bytes.
   - `pub fn private_bytes(&self) -> [u8; 32]` — `signing_key.to_bytes()`.
   - `pub fn public_bytes(&self) -> [u8; 32]` — `verifying_key.to_bytes()`.
   - `pub fn public_hex(&self) -> String` — `hex::encode(self.public_bytes())`.
   - `pub fn sign(&self, data: &[u8]) -> Signature` — `self.signing_key.sign(data)`.
   - `pub fn verify_signature(verifying_key: &VerifyingKey, data: &[u8], signature: &Signature) -> Result<(), BlufioError>` — static method, returns `BlufioError::Security` on failure.

2. `pub fn compute_content_hash(data: &[u8]) -> String` — SHA-256 digest, hex-encoded (64 chars).

3. File I/O for keypair persistence:
   - `pub fn save_keypair_to_file(keypair: &PublisherKeypair, private_path: &Path, public_path: &Path) -> Result<(), BlufioError>` — Write PEM-like format:
     ```
     -----BEGIN BLUFIO PUBLISHER PRIVATE KEY-----
     {hex-encoded 32 bytes}
     -----END BLUFIO PUBLISHER PRIVATE KEY-----
     ```
     And for public key:
     ```
     -----BEGIN BLUFIO PUBLISHER PUBLIC KEY-----
     {hex-encoded 32 bytes}
     -----END BLUFIO PUBLISHER PUBLIC KEY-----
     ```
   - `pub fn load_private_key_from_file(path: &Path) -> Result<PublisherKeypair, BlufioError>` — Parse PEM-like format, extract hex, decode to 32 bytes, reconstruct.
   - `pub fn load_public_key_from_file(path: &Path) -> Result<VerifyingKey, BlufioError>` — Parse PEM-like format, extract hex, decode to 32 bytes, construct VerifyingKey.

4. Signature serialization helpers:
   - `pub fn signature_to_hex(sig: &Signature) -> String` — hex-encode the 64-byte signature.
   - `pub fn signature_from_hex(hex_str: &str) -> Result<Signature, BlufioError>` — decode hex to 64 bytes, construct Signature.

**lib.rs:** Add `pub mod signing;` and re-export key public types: `pub use signing::{PublisherKeypair, compute_content_hash};`.

-- Add cryptographic verification columns to installed_skills.
ALTER TABLE installed_skills ADD COLUMN content_hash TEXT;

ALTER TABLE installed_skills ADD COLUMN publisher_id TEXT;

-- Publisher key store for TOFU (trust on first use) model.

    publisher_id TEXT PRIMARY KEY,

    first_seen TEXT NOT NULL,

```

Note: New columns in installed_skills are nullable (no NOT NULL) so existing rows are unaffected. NULL content_hash means hash not yet computed (legacy install). NULL signature means unsigned skill.

**Sign:** Read WASM bytes from wasm_path. Load private key via `load_private_key_from_file`. Compute content_hash. Sign the WASM bytes. Write output file `{wasm_path}.sig` containing:

signature={signature_hex}
```
Print success message with hash and publisher ID.

**Update:** Read WASM bytes and manifest, parse manifest, compute hash, check for .sig file next to WASM, call store.update() with optional signature data. Print success.
