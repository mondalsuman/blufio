      ```rust

      ```

   b. Update `ExternalTool::new()` to store `server_name`:
      ```rust
      pub fn new(
          server_name: &str,
          tool_name: String,
          description: String,
          schema: serde_json::Value,
          session: Arc<RunningService<RoleClient, ()>>,
          response_size_cap: usize,

              namespaced_name,
              description,
              schema,
              session,
              response_size_cap,
          }
      }
      ```

   c. Add a public getter for server_name:
      ```rust
      /// Get the server name for cost attribution.
      pub fn server_name(&self) -> &str {
          &self.server_name
      }
      ```
