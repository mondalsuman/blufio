  1. Add `mmr_rerank` function to retriever.rs (private, before `#[cfg(test)]`):
     ```rust
     fn mmr_rerank(scored: &[ScoredMemory], lambda: f64, k: usize) -> Vec<ScoredMemory>
     ```
     - Standard greedy MMR algorithm per Carbonell & Goldstein 1998
     - Operates on pre-sorted (by score descending) `scored` slice
     - First selection: always index 0 (highest score)
     - Iterative selection: for each remaining candidate, compute MMR score = `lambda * relevance - (1-lambda) * max_similarity_to_selected`
     - Uses `cosine_similarity()` from types.rs on `memory.embedding`
     - Returns cloned ScoredMemory vec of up to `k` items
     - Handle edge cases: empty input, k=0, k > len, single item
