```rust
/// Persists pre-extracted entity strings as Memory entries with MemorySource::Extracted.
///
/// Called by SessionActor after context assembly when compaction extracts entities.
/// Each entity string is embedded and saved to the MemoryStore with 0.6 confidence
/// (lower than explicit memories).
pub async fn persist_extracted_entities(
    &self,
    session_id: &str,
    entities: &[String],
) -> Result<usize, BlufioError> {
```

Implementation:
- Return early with 0 if `entities` is empty
- For each entity string:
  - Generate embedding via `self.embedder.embed(EmbeddingInput::Single(entity.clone())).await`
  - If embedding fails, log warning and skip (non-fatal)
  - Create a `Memory` struct with:
    - `id`: `uuid::Uuid::new_v4().to_string()`
    - `content`: the entity string
    - `embedding`: first vector from embedding result (or skip if empty)
    - `source`: `MemorySource::Extracted`
    - `confidence`: `0.6` (extracted entities are less confident than explicit)
    - `status`: `MemoryStatus::Active`
    - `superseded_by`: `None`
    - `session_id`: `Some(session_id.to_string())`
    - `classification`: `DataClassification::default()`
    - `created_at` / `updated_at`: `chrono::Utc::now().to_rfc3339()`
  - Call `self.store.save(&memory).await`
  - If save fails, log warning and continue (non-fatal, best-effort)
  - Increment saved counter
- Return the count of successfully saved entities
- Log at info level: "persisted {count} extracted entities from compaction"

**Step 3: Call persist_extracted_entities in SessionActor (crates/blufio-agent/src/session.rs)**

```rust
// Persist entities extracted during compaction as Memory entries (COMP-06).
if !assembled.extracted_entities.is_empty() {
    if let Some(ref extractor) = self.memory_extractor {
        match extractor
            .persist_extracted_entities(&self.session_id, &assembled.extracted_entities)
            .await
        {
            Ok(count) => {
                if count > 0 {
                    info!(
                        session_id = %self.session_id,
                        count = count,
                        "persisted extracted entities from compaction"
                    );
                }
            }
            Err(e) => {
                warn!(
                    session_id = %self.session_id,
                    error = %e,
                    "failed to persist extracted entities (non-fatal)"
                );
            }
        }
    }
}
```
