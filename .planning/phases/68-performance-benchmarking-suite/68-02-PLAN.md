2. **Setup function:** Create `setup_hybrid_bench(count: usize)` that:
   - Creates an in-memory SQLite DB with memories table + vec0 virtual table
   - Generates `count` memories with realistic text content (short sentences about different topics for BM25 diversity)
   - Embeds each memory using the ONNX model (real embeddings, not deterministic fakes)
   - Inserts into both memories table and vec0
   - Returns the DB connection + a query embedding (embed a search query like "What did we discuss about the project?")
