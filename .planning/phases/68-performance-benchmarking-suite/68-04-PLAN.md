  <action>
Add a new `binary-size` job to bench.yml (runs in parallel with the benchmark job).

**binary-size job:**
1. `runs-on: ubuntu-latest`
2. `timeout-minutes: 15`
3. Steps:
   - `actions/checkout@v4`
   - Install system deps (same as benchmark job: `sudo apt-get update && sudo apt-get install -y libssl-dev`)
   - `dtolnay/rust-toolchain@stable`
   - `Swatinem/rust-cache@v2` with `shared-key: binary-size`
   - Install cargo-bloat: `cargo install cargo-bloat --locked`

4. **Build release binary:**
   ```yaml
   - name: Build release binary
     run: cargo build --release -p blufio
   ```

5. **Measure binary size:**
   ```yaml
   - name: Measure binary size
     run: |
       SIZE=$(stat -c%s target/release/blufio 2>/dev/null || stat -f%z target/release/blufio)
       SIZE_MB=$(echo "scale=2; $SIZE / 1048576" | bc)
       echo "Binary size: ${SIZE_MB}MB ($SIZE bytes)"
       echo "binary_size=$SIZE" >> $GITHUB_OUTPUT
       echo "binary_size_mb=$SIZE_MB" >> $GITHUB_OUTPUT

       # Warn at 50MB
       if [ "$SIZE" -gt 52428800 ]; then
         echo "::warning::Binary size ${SIZE_MB}MB exceeds 50MB target"
       fi

       # Fail at 55MB
       if [ "$SIZE" -gt 57671680 ]; then
         echo "::error::Binary size ${SIZE_MB}MB exceeds 55MB hard limit"
         exit 1
       fi

       echo "Binary size ${SIZE_MB}MB is within limits"
     id: size
   ```

6. **Per-crate breakdown:**
   ```yaml
   - name: Per-crate size breakdown
     run: cargo bloat --release -p blufio --crates -n 20
   ```

7. **Feature-flag comparison (sqlite-vec delta):**
   Read `crates/blufio-memory/Cargo.toml` to find the correct feature flag for sqlite-vec. Build without it and compare:
   ```yaml
   - name: Build without sqlite-vec (size comparison)
     run: |
       # Build with all default features except sqlite-vec
       # Exact flags depend on feature structure -- executor must read Cargo.toml
       cargo build --release -p blufio --no-default-features --features "..." 2>/dev/null || echo "Feature-flag build failed -- sqlite-vec may not be feature-gated"
     continue-on-error: true

   - name: Report sqlite-vec size delta
     if: success()
     run: |
       FULL_SIZE=$(stat -c%s target/release/blufio 2>/dev/null || stat -f%z target/release/blufio)
       # Compare with previous build if available
       echo "Full binary (with sqlite-vec): ${{ steps.size.outputs.binary_size_mb }}MB"
       echo "sqlite-vec contributes to the total binary size"
   ```

8. **Debug binary size (for tracking):**
