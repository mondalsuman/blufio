**Cargo.toml:** Dependencies: blufio-core, blufio-config (path), async-trait, tokio (workspace), serde, serde_json, tracing, semver, uuid (workspace). Pin matrix-sdk exactly: `matrix-sdk = { version = "=0.11.0", default-features = false, features = ["rustls-tls"] }`. Do NOT enable `e2e-encryption` feature (deferred to EXT-06). This avoids pulling in matrix-sdk-crypto and its heavy dependency tree.

**handler.rs:** Implement event handler functions as free functions (not closures, following matrix-sdk pattern):

1. `pub async fn on_room_message(event: OriginalSyncRoomMessageEvent, room: Room, inbound_tx: Ctx<mpsc::Sender<InboundMessage>>, bot_user_id: Ctx<String>, allowed_users: Ctx<Vec<String>>)`:
   - Skip if room state is not Joined.
   - Extract text from event.content.msgtype (match MessageType::Text, skip others).
   - Skip messages from the bot itself (event.sender == bot_user_id).
   - If allowed_users is non-empty, check event.sender is in allowed list.
   - Check for @mention of bot display name or user ID in message text. In DM rooms (room members count <= 2), always process. In group rooms, only process if @mentioned.
   - Create InboundMessage with id=event.event_id.to_string(), channel="matrix", sender_id=event.sender.to_string(), content=text body (stripped of @mention), metadata=json({"chat_id": room.room_id().to_string(), "sender_name": event.sender.localpart()}).
   - Send via inbound_tx. Log warning if channel full.

2. `pub async fn on_room_invite(event: StrippedRoomMemberEvent, client: Client, room: Room)`:
   - Check if the invite is for us: event.state_key == client.user_id().
   - If yes, spawn a task that retries room.join() up to 3 times with 2-second delays between attempts. Log success or failure.

NOTE on matrix-sdk 0.11 handler registration: Handlers receive context values via the `Ctx<T>` type (or equivalent). The executor should check the exact matrix-sdk 0.11 API for registering handlers with context. The pattern is typically:
```rust
client.add_event_handler_context(inbound_tx.clone());
client.add_event_handler_context(bot_user_id.clone());
client.add_event_handler(on_room_message);
client.add_event_handler(on_room_invite);
```

**lib.rs:** Implement `MatrixChannel` struct:
- Fields: config (MatrixConfig), inbound_rx (Mutex<mpsc::Receiver<InboundMessage>>), inbound_tx (mpsc::Sender<InboundMessage>), client (Option<matrix_sdk::Client>), sync_handle (Option<JoinHandle<()>>).
- `pub fn new(config: MatrixConfig) -> Result<Self, BlufioError>`: Validate homeserver_url, username, and password are set. Create mpsc channel(100).
- PluginAdapter: name "matrix", version 0.1.0, adapter_type Channel. health_check: if client exists, return Healthy; else Unhealthy. shutdown: abort sync_handle if exists.
- ChannelAdapter capabilities: supports_edit: true (Matrix supports message editing via m.replace), supports_typing: true, supports_images: true, supports_documents: true, supports_voice: false, max_message_length: Some(65536), supports_embeds: false, supports_reactions: true, supports_threads: true.
- `connect()`:
  1. Build client via `Client::builder().homeserver_url(&config.homeserver_url).build().await?`.
  2. Login: `client.matrix_auth().login_username(&config.username, &config.password).initial_device_display_name("Blufio").send().await?`.
  3. Store bot user ID from `client.user_id()`.
  4. Register event handlers via add_event_handler_context + add_event_handler for on_room_message and on_room_invite.
  5. If config.rooms is non-empty, join each room via `client.join_room_by_id_or_alias()` (log errors, don't fail on individual room join failures).
  6. Spawn sync task: `tokio::spawn(async move { client_clone.sync(SyncSettings::default()).await; })`. The sync runs indefinitely processing events.
  7. Store client and sync_handle.
- `send()`: Extract room_id from metadata chat_id. Get room via `client.get_room(&room_id)`. Call `room.send(RoomMessageEventContent::text_plain(&msg.content)).await`. Return MessageId with event ID.
- `receive()`: Read from inbound_rx.
- `edit_message()`: Matrix supports edits via replacement events. Use `RoomMessageEventContent::text_plain(text).make_replacement(original_event_id)` pattern. Get room, send replacement.
- `send_typing()`: Call `room.typing_notice(true).await`. Return Ok(()).

```rust
#[cfg(feature = "matrix")]
use blufio_matrix::MatrixChannel;

// In run_serve(), after IRC wiring:
#[cfg(feature = "matrix")]
{
    if config.matrix.homeserver_url.is_some() {
        let matrix = MatrixChannel::new(config.matrix.clone()).map_err(|e| {
            error!(error = %e, "failed to initialize Matrix channel");
            eprintln!(
                "error: Matrix homeserver URL, username, and password required. \
                 Set in [matrix] config section"
            );
            e
        })?;
        mux.add_channel("matrix".to_string(), Box::new(matrix));
        info!("matrix channel added to multiplexer");

        // Redact Matrix password in logs.
        if let Some(ref password) = config.matrix.password {
            blufio_security::RedactingWriter::<std::io::Stderr>::add_vault_value(
                &vault_values,
                password.clone(),
            );
        }
    } else {
        info!("matrix channel skipped (no homeserver_url configured)");
    }
}
```
