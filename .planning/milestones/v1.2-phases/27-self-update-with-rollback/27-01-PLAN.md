   - `async fn fetch_latest_release() -> Result<ReleaseInfo, BlufioError>` -- GET `{API_BASE}/repos/{GITHUB_REPO}/releases/latest` with User-Agent header `blufio/{version}` and Accept `application/vnd.github+json`. Handle non-success status codes (especially 403 for rate limit, 404 for no releases). Parse response as GitHubRelease, call find_asset().
   - `async fn download_to_temp(url: &str, dir: &Path) -> Result<tempfile::NamedTempFile, BlufioError>` -- download binary content with reqwest, write to NamedTempFile created in `dir` (same filesystem as binary for later rename). Log size to stderr after download.
   - `fn verify_download(binary_path: &Path, sig_path: &Path) -> Result<(), BlufioError>` -- call `blufio_verify::verify_signature(binary_path, Some(sig_path))`, map VerifyError to BlufioError::Update.

   **Public entry points (wired in Plan 02):**

   ```rust
   /// Update subcommands.
   #[derive(Subcommand, Debug)]
   enum UpdateCommands {
       /// Check for available updates without downloading.
       Check,
       /// Rollback to the pre-update binary.
       Rollback,
   }
   ```

3. Add `Update` variant to `Commands` enum:
   ```rust
   /// Update Blufio to the latest version.
   Update {
       #[command(subcommand)]
       action: Option<UpdateCommands>,
       /// Skip interactive confirmation.
       #[arg(long)]
       yes: bool,
   },
   ```

4. Add match arm in main():
   ```rust
   Some(Commands::Update { action, yes }) => match action {
       Some(UpdateCommands::Check) => {
           if let Err(e) = update::run_check().await {
               eprintln!("error: {e}");
               std::process::exit(1);
           }
       }
       Some(UpdateCommands::Rollback) => {
           if let Err(e) = update::run_rollback() {
               eprintln!("error: {e}");
               std::process::exit(1);
           }
       }
       None => {
           if let Err(e) = update::run_update(yes).await {
               eprintln!("error: {e}");
               std::process::exit(1);
           }
       }
   },
   ```
   NOTE: `run_rollback()` will be created in Plan 02. For now, create a placeholder:
   ```rust

       Err(BlufioError::Update("rollback not yet implemented".into()))
   }
   ```
