# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |

Only the latest release receives security updates.

## Reporting a Vulnerability

**Do NOT open a public issue for security vulnerabilities.**

Report vulnerabilities through [GitHub Security Advisories](https://github.com/blufio/blufio/security/advisories/new):

1. Go to **Settings > Security > Advisories**
2. Click **New draft security advisory**
3. Fill in the details and submit

## Response Timeline

- **Acknowledgment:** Within 48 hours
- **Initial assessment:** Within 7 days
- **Fix timeline:** Within 90 days of report

## Disclosure

We follow a 90-day coordinated disclosure timeline. After a fix is released,
we will publish a security advisory with details of the vulnerability.
